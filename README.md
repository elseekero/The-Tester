  IN THE DEAD OF THE PAIN - DOOM TEST MAP
==============================================

Be sure you know a little from Doom!

Requirements:
- GZDoom
- The WAD
- A life
- A wife
- And family

==============================================
HOW TO USE THE TEST MAP
==============================================

This is a sandbox map for testing weapons and enemies.

1. Enable cheats first:
   Open console (`) and type: sv_cheats 1

2. Spawn enemies:
   Open console (`) and type: summon [enemy name]
   Examples:
   - summon zombieman
   - summon imp
   - summon cacodemon
   - summon cyberdemon

3. Get weapons:
   Open console (`) and type: give [weapon/item]
   Examples:
   - give shotgun
   - give chaingun
   - give rocketlauncher
   - give all (gives everything)

4. Other useful commands:
   - god (invincibility)
   - noclip (walk through walls)
   - kill monsters (clear all enemies)
   - resurrect (respawn if you die)
